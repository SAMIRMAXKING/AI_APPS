الخطة 133: مكتبة تعليمية مدمجة للتعلم من داخل التطبيق (Interactive Learning)

━━━━━━━━━━━━━━━━━━━━━━━━━━━
■ الهدف:
دمج مكتبة تفاعلية تعليمية داخل التطبيق تحتوي على دروس مصغرة، اختبارات، مشاريع مصغرة وتمارين قابلة للتطبيق الفوري لتحسين مستوى المستخدم في البرمجة، الأدوات، الذكاء الاصطناعي، وتصميم التطبيقات.

━━━━━━━━━━━━━━━━━━━━━━━━━━━
■ المكونات الأساسية:
1. واجهة عرض تعليمية مخصصة (تعليم مصغر).
2. تقسيم المحتوى إلى أقسام ودورات.
3. دعم تشغيل الأكواد التجريبية مباشرة داخل التطبيق.
4. اختبار سريع بعد كل وحدة.
5. شارات/نقاط تحفيزية للتفاعل.
6. دعم المحتوى باللغة العربية والإنجليزية.
7. إمكانية تحديث الدروس تلقائيًا عبر الإنترنت.

━━━━━━━━━━━━━━━━━━━━━━━━━━━
■ التفاصيل البرمجية الدقيقة:

1. واجهة التعليم:
   - قسم رئيسي: [🏫 تعلم] في الشريط الجانبي
   - عرض الوحدات: [مقدمة بايثون، إدارة مشاريع، Git، Debugging…]

2. تشغيل الأكواد:
   - محرر مصغر داخل صفحة الدرس
   - زر [جرب الكود] لتنفيذه مباشرة

3. الاختبارات:
   - أسئلة اختيار من متعدد، صح/خطأ، أكواد ناقصة
   - زر [تحقق من النتيجة]
   - سجل التقدم

4. النقاط والشارات:
   - كل درس = نقاط
   - بعد كل 5 دروس → شارة

5. التحديث:
   - جلب محتوى JSON عبر الإنترنت
   - دعم تحديث المحتوى من GitHub أو من API خارجي
